# Django To-Do List Web Application

This is a simple web application built using Django that allows users to manage their to-do tasks. Users can register, log in, add tasks, mark tasks as completed, and delete tasks.

## Features

- User registration and authentication
- Create, view, edit, and delete tasks
- Mark tasks as completed
- Responsive design for desktop and mobile devices
